Eye-Tracking Text Summarization System


This project is created for summarizing text based on the eye movements of the participants.
The experiment includes 5 steps:
1. Text reading
2. Text summarization
3. Sentences highlighting
4. Sentences ranking
5. Questions answering


Final Project, SISE 2021
